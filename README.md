## Dependencies
The project requries `rustc` (rust compiler) to work.
It's however, better to use `cargo` (rust package manager) to compile the program.
To install dependencies in linux run the below commands:
```
sudo apt install cargo rustc # Debian/Ubuntu
sudo dnf install cargo rustc # Fedora
sudo pacman -S cargo rustc # Arch/Manjaro
```
Installing dependencies in Windows is kind of complicated
but since you know rust, you know how to install rust dependencies.
